Install dalam 4 langkah

Tidak perlu Chrome Web Store. Install langsung via Developer Mode, gratis.

01 —

Download file ZIP

Klik tombol Download Extension. File .zip akan tersimpan di folder Downloads.

02 —

Ekstrak ZIP

Klik kanan → Extract All. Simpan folder hasil ekstrak di tempat yang mudah — jangan dihapus.

03 —

Buka halaman Extensions

Buka Chrome → ketik chrome://extensions → aktifkan Developer mode di pojok kanan atas.

04 —

Load unpacked

Klik Load unpacked → pilih folder hasil ekstrak. Extension langsung aktif di toolbar.

Fitur

Semua yang kamu butuhkan

Dirancang khusus untuk UI Sakem versi 2025+.

Auto Archive

Centang, klik arsip, konfirmasi dialog — semua berjalan otomatis tanpa intervensi.

Counter Akurat

Processed & Remaining dari paginator "dari X" — tidak bisa salah tab.

Auto Recovery

Stuck? Extension otomatis retry setelah 5 cycle tanpa progres.

Custom Delay

Atur jeda antar aksi (ms). Default 2000ms, bisa dikurangi sesuai koneksi.

Activity Log

Log berwarna real-time tiap langkah: centang, klik, simpan, hasil batch.

Stage Indicator

3 bar menunjukkan tahap yang sedang berjalan: centang → arsip → konfirmasi.
